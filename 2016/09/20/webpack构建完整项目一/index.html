webpack构建完整项目（一）

Sep 20, 2016 | webpack | Hits

说明

根据自己项目的需求，搭建一个完整webpack项目，包含的功能有：

  • js打包
  • js压缩
  • css编译（sass，自动前缀）
  • css压缩
  • js和css分离
  • 提取多个入口文件的公共脚本部分，生成公共js，方便多页面之间的复用
  • 独立打包样式文件
  • 图片压缩
  • 字体
  • 使用VUE

安装

1.安装webpack
npm init

npm install –save-dev webpack

2.新建目录

3.配置webpack.config.js

4.安装开发服务器实时监测
npm install webpack-dev-server –save-dev

5.配置package.json，可以使用快捷命令
npm start
npm run dev
npm run build

6.运行
